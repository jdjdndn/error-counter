# Error Counter - VSCode 错误计数器

一个多功能错误显示插件，**独有的文件夹错误聚合计数**是其他同类插件没有的功能。

## 核心亮点

### 🎯 文件夹错误聚合（独有功能）
其他插件只能显示单个文件的错误数，**本插件支持文件夹递归聚合**：
- 自动计算文件夹下所有子文件的错误总数
- 实时更新，无需手动刷新
- 基于前缀树（Trie）算法，O(1) 查询复杂度

```
📁 src/                    → 5  (聚合所有子文件错误)
  📁 components/           → 3
    📄 Button.tsx          → 1
    📄 Modal.tsx           → 2
  📁 utils/                → 2
    📄 helper.ts           → 2
  📄 index.ts              → 0
```

## 功能详情

### 1. 文件树错误徽章
- 文件名后显示错误/警告数量
- 纯数字显示：`2`（红色错误）、`3`（黄色警告）
- 同时存在：`2/3`

### 2. 文件夹错误聚合（独有）
- 自动递归计算子文件夹错误
- 实时增量更新
- 前缀树算法优化，大项目也流畅
- **显示方式**：
  - 文件：显示数字徽章（如 `2`）
  - 文件夹：显示红色圆点指示器，**鼠标悬停查看具体错误数**
  - > ⚠️ 由于 VSCode API 限制，文件夹无法显示数字徽章，但可以通过悬停或侧边栏面板查看聚合错误数

### 3. 状态栏显示
- 左下角显示：`✓ 0` 或 `⊗ 5 ⚠ 3`
- 点击打开问题面板
- 实时更新

### 4. 代码行内提示（Error Lens）
- 在错误行尾显示详细信息
- 示例：`const x: string = 123; // 类型错误`
- 可替代 Error Lens 插件

### 5. 错误概览面板
- 左侧活动栏显示错误图标
- 展开查看所有文件错误
- 按错误数排序
- 点击跳转到错误位置
- 右键复制错误信息

### 6. 快速导航
| 快捷键 | 功能 |
|--------|------|
| `Alt+N` | 跳转到下一个错误 |
| `Alt+P` | 跳转到上一个错误 |

### 7. 错误通知
- 错误数超过 10 个时提醒
- 单次新增 5+ 错误时警告
- 可点击查看问题面板

### 8. 导出错误报告
- 生成 Markdown 格式报告
- 保存到项目根目录 `error-report.md`
- 包含文件路径、行号、错误信息

### 9. 忽略配置

**settings.json：**
```json
{
  "errorCounter.ignorePatterns": [
    "**/node_modules/**",
    "**/dist/**",
    "**/*.d.ts"
  ]
}
```

**.errorignore 文件：**
```gitignore
# 依赖
node_modules/

# 构建输出
dist/
build/

# 生成文件
*.d.ts
*.min.js
```

## 性能优化

### 前缀树（Trie）算法

使用前缀树存储文件路径和错误计数：

| 操作 | 传统方式 | 前缀树 |
|------|----------|--------|
| 查询文件夹聚合 | O(n) | **O(1)** |
| 更新单个文件 | O(n) | **O(k)** |

> k = 路径深度（<10），n = 文件总数（可能 >10000）

### 其他优化
- 增量更新：只处理变化的文件
- 差分计算：只计算新旧值差值
- LRU 缓存：缓存路径解析结果
- 防抖机制：300ms 防抖避免频繁更新

### 性能对比

| 项目规模 | 文件数 | 扫描时间 | 内存占用 |
|----------|--------|----------|----------|
| 小型 | <1000 | <50ms | ~1MB |
| 中型 | 1000-10000 | <200ms | ~5MB |
| 大型 | >10000 | <500ms | ~15MB |

## 安装

### 从源码安装
```bash
git clone <repo>
cd error-counter
npm install
npm run compile
```
按 F5 在 VSCode 中调试

### 打包安装
```bash
npm install -g @vscode/vsce
vsce package
```
安装生成的 `.vsix` 文件
